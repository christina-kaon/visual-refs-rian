Q/C/A 三层每个子维度的用户需求大盘

基于 80,000 prompt sample (5/15-5/16) keyword 占比 + top100 bot trope 分布 · 给迭代优先级提供数据 anchor
样本基线：从 rds_pg.prompt_image_history 拉 2 天 80,000 个 SUCCESS chat-scenario 生图样本（8 个 6h 窗口避 PG recovery conflict）。Anime 69.8% (55,808) / Realistic 30.2% (24,141)。

子维度占比统计方法：对每个子维度的 keyword set 取 OR 匹配，分别在 anime / real sub-population 内统计占比。"信号"分级：≥5% 强 / 1-5% 中 / 0.1-1% 弱 / 0% 无 prompt-level 信号（不等于无 demand，见下 caveat）。

数据 caveat（很重要）

  1. emochi prompt 是经过 prompt rewriting / wrapping 的，不是用户原始描述。"cinematic" 99.72% 真人 prompt 含 = emochi 后端模板自动注入。"wide shot" 44% real 也类似。所以 keyword 占比不等于用户主动需求强度
  2. Aesthetics 流派关键词大多 ≈ 0%（kawaii / y2k / cel shading / genshin / spider-verse 等）— 这是因为这些流派是视觉特征而非 prompt 关键词。用户不会在剧情 prompt 里写"genshin style"，但这不代表他们不喜欢这种风格
  3. Bot trope 分布更准：top100 bot 的 trope 分布（marriage 占 anime 15% / OC Real 占 real 52%）能间接反映用户对 Aesthetics 流派的潜在 demand
  4. 这份矩阵的正确用法：
    • Content 层信号 ≥ 1% = 明显用户主动表达 → 必须覆盖
    • Aesthetics 层用 top100 bot trope + 已知 cross-validation（如 BTS 9 个进 real top100 → K-pop 美学有真实 demand）
    • 0% keyword 不等于 0 需求 — 反过来需要"用户主动用 prompt 写"作为正向 evidence，而不是反向证伪

Quality 层（5 维度）

⚠️ Quality 维度 prompt-level keyword 反映不出"失败率" — 因为用户不会在 prompt 里写"请不要糊"。Quality 优先级靠 perceptual heuristic 排：解剖学崩 > 糊 > 语义错 > AI 感 > 风格撕裂。5 个都是 P0 / P1 — 不要降级，是 gate condition

维度优先级WHY训练方法
D1 整体糊P0基础门槛 — 不过 reject 训练集直接 reject sampling
D2 恐怖谷 / 肢体扭曲P0基础门槛 — 真人比动漫敏感度更高face anatomy 专项数据 + 反例 dataset
D3 语义崩坏P0基础门槛 — 多人交互 / 物理一致性multi-person + holding interaction 专项
D4 塑料 AI 感P1主观感受 — 真人皮肤纹理 / 动漫线条质感皮肤/线条质感 LoRA + 磨皮反例剔除
D5 风格撕裂P1主观感受 — character + scene 统一光照端到端 character-scene 训练

v1.0 迭代优先级总结（综合 prompt 信号 + bot trope + 已知 cross-validation）

P0（立即做）：

  • Quality 5 维度全部 P0（gate condition）
  • Content · 真人：embrace 10% / bedroom 31% / rain night 11%（real）/ kissing 2.5%（anime）/ low_angle 9.7%（anime） — 都是 ≥1% 强信号
  • Content · 动漫：magic_cast 15.7% / smirk 5% / kneeling 5.5% / blush 3% — 都是 ≥1% 强信号
  • Aesthetics · 真人 B Dark Romance：cinematic 99.7% baseline + possessive 11% + ceo 0.8% + mafia 0.35% → P0 (用户 prompt 主动量级最大)
  • Aesthetics · 动漫 marriage/OC Virtual 系（bot 51% top100）— 即使 prompt-level keyword 弱（otome 0% / bishounen 0.01%），bot trope demand 是数据 anchor

P1（次优先 / 扩量储备）：

  • Content · 真人：princess_carry 0.1% / forehead_touch 0.14% — 用户写少但 dark romance 经典 trope，作为风格 marker 训
  • Aesthetics · 真人 A 日韩氧气：natural_light 0.7% / k_idol 0.03% — prompt 弱但 BTS 9/100 top bot 数据强 → P1
  • Aesthetics · 真人 C 抖红美颜：v_line 3% real 内（间接 signal）— P1，等东亚扩量

P2 / 砍：

  • 真人 3A 游戏建模：3d_render / unreal / octane / concept_art 全部 0% — 确认砍（仅作 base pretrain 训练集填充物，原则 6 已说明）
  • 真人高清艺术摄影：editorial 0.14% / magazine 0.16% / softbox 0.07% — 确认 P2（同上，作填充物）
  • 动漫游戏 2.5D / 美漫 2D：spider_verse / hazbin 等 0-0.05% — 长尾兜底
